# PowerBI_MobileSales_Dashboard

https://github.com/user-attachments/assets/fc564333-645d-4227-8d5d-af4033e336bd

This repository contains an **Interactive Mobile Sales Dashboard** developed using **Power BI Desktop**.  
The dashboard provides meaningful insights into mobile sales performance using dynamic visuals and slicers.

---

## 🚀 Project Overview

The goal of this project is to analyze mobile sales data and present it in a visually appealing and interactive way to support **business decision-making**.

This project was created as part of my learning from **Satish Dhawale’s Skill Course**, focusing on practical and industry-relevant Power BI skills.

---

## 🔍 Key Insights Covered

### ✅ Key Performance Indicators (KPIs)
- **Total Sales**
- **Total Quantity Sold**
- **Total Transactions**
- **Average Price**

### 📊 Visualizations Included
- Total Quantity by Month (Trend Analysis)
- Sales Distribution by City (Map Visual)
- Transactions by Payment Method
- Customer Ratings Analysis
- Total Sales by Mobile Model
- Day-wise Sales Trend

---

## 🎛️ Interactive Features
- Slicers for:
  - Month
  - Mobile Model
  - Brand
  - Payment Method
- Drill-down & cross-filtering across visuals

---

## 🛠️ Tools & Skills Used
- Power BI Desktop
- Data Cleaning & Modeling
- DAX Measures
- KPI Cards
- Slicers & Filters
- Map & Chart Visuals
- Dashboard UI Design
